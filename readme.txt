== Description ==

The plugin is saved as a widget on your Wordpress blog for inserting external HMTL codes. Thanks to this feature, you can add it to your blog via drag and drop in any theme.

* Adds two widgets to your Wordpress blog. (Runive HTML Box 1 and 2)
* You can use choose one or use both widgets together.
* It is compatible with all themes. (As far as it supports widgets)
* Supports all standart HTML and JavaScript codes.
* Also supports all advertisement programs code such as Google AdSense 

== Installation ==

You can use the built in installer and upgrader, or you can install the plugin
manually.

1. Upload files to the `/wp-content/plugins/runivehtmlbox/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to widgets and drag and drop `Runive HTML Box 1 or 2` to Sidebar and configure it.
